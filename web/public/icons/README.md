## Como Gerar os Ícones

### Opção 1: Usando PWA Builder (Recomendado)
1. Acesse: https://www.pwabuilder.com/
2. Insira a URL do seu site
3. Baixe o pacote de ícones gerado
4. Extraia os arquivos para este diretório

### Opção 2: Usando ferramentas online
- **Favicon Generator**: https://realfavicongenerator.net/
- **PWA Icon Generator**: https://tools.crawlink.com/tools/pwa-icon-generator/

### Opção 3: Usando ferramentas de design
- Adobe Illustrator/Photoshop
- Figma
- Canva

## Especificações Técnicas

### Formato
- **Formato**: PNG
- **Fundo**: Transparente ou sólido (preferencialmente branco)
- **Qualidade**: Alta resolução

### Design Guidelines
- **Estilo**: Consistente com a identidade visual do SpeechAI
- **Cores**: Usar a paleta de cores do tema (#3b82f6)
- **Simplicidade**: Design limpo e reconhecível em tamanhos pequenos
- **Contraste**: Garantir boa visibilidade em diferentes fundos

### Ícone Principal
O ícone principal deve representar:
- Microfone ou ondas sonoras (relacionado à fala)
- IA ou elementos tecnológicos
- Identidade visual do SpeechAI

## Verificação
Após adicionar os ícones, verifique se:
1. Todos os arquivos estão no formato correto
2. As dimensões estão corretas
3. Os ícones são legíveis em diferentes tamanhos
4. O PWA funciona corretamente no navegador
